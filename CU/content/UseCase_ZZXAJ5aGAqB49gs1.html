Escenario 1: Crear
Steps
1. Lider da click en el botón administrar las vinculaciones en el menú
2. SYSTEM abre un modulo con todas las vinculaciones existentes
3. Lider selecciona la opción administrar las vinculaciones
4. SYSTEM abre una ventana donde aparecen todas las vinculaciones
5. Lider crea la vinculación y la guarda
6. SYSTEM valida y crea la vinculación

Escenario 2: Modificar
Steps
1. Lider da click en el botón administrar las vinculaciones en el menú
2. SYSTEM abre un modulo con todas las vinculaciones existentes
3. Lider selecciona la opción administrar las vinculaciones
4. SYSTEM abre una ventana donde aparecen todas las vinculaciones
5. Lider modifica la vinculación y la guarda
6. SYSTEM valida y modifica la vinculación

Escenario 3: Eliminar
Steps
1. Lider da click en el botón administrar las vinculaciones en el menú
2. SYSTEM abre un modulo con todas las vinculaciones existentes
3. Lider selecciona la opción administrar las vinculaciones
4. SYSTEM abre una ventana donde aparecen todas las vinculaciones
5. if confirmación de la eliminación
    5.1. Lider confirma la eliminación, vuelve al modulo administrar las vinculaciones
    5.2. SYSTEM elimina la vinculación
6. else if
    6.1. Lider da click en cancelar eliminación
    6.2. SYSTEM cancela la eliminación y vuelve al modulo administrar las vinvulaciones
   end if
