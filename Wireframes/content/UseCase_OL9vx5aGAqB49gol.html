escenario-eliminar

2. SYSTEM abre un modulo con todos los ambientes existentes
abre un modulo con todos los ambientes existentes

4. SYSTEM abre una ventana para confirmar la eliminación
5. if confirmación de la eliminación

6. else if
    6.1. Lider da click en cancelar la eliminación
    6.2. SYSTEM cancela la eliminación y vuelve al modulo de administrar los ambientes
   end if
