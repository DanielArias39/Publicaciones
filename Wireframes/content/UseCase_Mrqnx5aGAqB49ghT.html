Escenario 1: crear
Steps
1. Lider selecciona el botón de gestionar las fichas, va al botón y le da crear ficha
2. SYSTEM abre un cuadro con todas las opciones que tiene para crear una ficha
abre un cuadro con todas las opciones que tiene para crear una ficha
3. Lider selecciona las opciones necesarias para la creación de una ficha, da en el botón guardar
4. SYSTEM se evalúan las opciones seleccionadas y da por terminado el proceso de crear una ficha

Extension
Steps
4.a. botón de cancelar
    1. Lider cancela el proceso
    2. SYSTEM se devuelve a la pagina principal de gestionar fichas para comenzar nuevamente la creación

Escenario 2: modificar
Steps
1. Lider selecciona el botón de gestionar las fichas, va al botón y le da modificar ficha
2. SYSTEM abre un cuadro con todas las opciones que tiene para modificar la ficha
abre un cuadro con todas las opciones que tiene para modificar la ficha
3. Lider selecciona las opciones necesarias para la modificación de una ficha, da en el botón guardar
4. SYSTEM se evalúan las opciones seleccionadas y da por terminado el proceso de modificar la ficha

Extension
Steps
4.a. botón de cancelar
    1. Lider cancela el proceso
    2. SYSTEM se devuelve a la pagina principal de gestionar las fichas para comenzar nuevamente la modificación

Escenario 3: eliminar
Steps
1. Lider selecciona en el botón gestionar fichas en el menú
2. SYSTEM abre un modulo con todos las fichas existentes existente
abre un modulo con todos las fichas existentes existente
3. Lider selecciona una ficha y selecciona en el botón eliminar
4. SYSTEM abre una ventana para confirmar la eliminación
5. if confirmación de la eliminación
    5.1. Lider confirma la eliminación, vuelve al modulo de gestionar las fichas
    5.2. SYSTEM elimina la ficha
6. else if
    6.1. Lider selecciona en cancelar la eliminación
    6.2. SYSTEM cancela la eliminación y vuelve al modulo de gestionar las fichas
   end if

Escenario 4: consultar
Steps
1. Lider selecciona en el botón gestionar fichas en el menú
2. SYSTEM abre un modulo con todas las fichas existente
abre un modulo con todas las fichas existente
3. Lider selecciona en el botón consultar
4. SYSTEM abre una ventana donde se sólita las opciones para buscar la ficha a su gusto
5. Lider dígita los datos correspondientes y selecciona en consultar
6. SYSTEM valida y busca la ficha

Extension
Steps
6.a. botón cancelar
    1. Lider cancela el proceso
    2. SYSTEM muestra la interfaz del modulo gestionar ficha
