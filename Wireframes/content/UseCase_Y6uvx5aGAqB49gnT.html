Escenario 1: crear
Steps
1. Lider selecciona el botón de gestionar las sedes del proyecto, va al botón y le da crear sede
2. SYSTEM abre un cuadro con todas las opciones que tiene para crear sede
abre un cuadro con todas las opciones que tiene para crear sede
3. Lider selecciona las opciones necesarias para la creación de una sede, da en el botón guardar
4. SYSTEM se evalúan las opciones seleccionadas y da por terminado el proceso de crear una sede

Extension
Steps
4.a. botón de cancelar
    1. Lider cancela el proceso
    2. SYSTEM se devuelve a la pagina principal de gestionar las sedes para comenzar nuevamente la creación

Escenario 2: modificar
Steps
1. Lider selecciona el botón de gestionar las sedes, va al botón y le da modificar sedes
2. SYSTEM abre un cuadro con todas las opciones que tiene para modificar las sedes
abre un cuadro con todas las opciones que tiene para modificar las sedes
3. Lider selecciona las opciones necesarias para la modificación de una sede, da en el botón guardar
4. SYSTEM se evalúan las opciones seleccionadas y da por terminado el proceso de modificar las sedes

Extension
Steps
4.a. botón de cancelar
    1. Lider cancela el proceso
    2. SYSTEM se devuelve a la pagina principal de gestionar las sedes para comenzar nuevamente la modificación

Escenario 3: eliminar
Steps
1. Lider selecciona en el botón gestionar las sedes en el menú
2. SYSTEM abre un modulo con todas las sedes existentes
abre un modulo con todas las sedes existentes
3. Lider selecciono una sede y selecciona en el botón eliminar
4. SYSTEM abre una ventana para confirmar la eliminación
5. if confirmación de la eliminación
    5.1. Lider confirma la eliminación, vuelve al modulo de gestionar las sedes
    5.2. SYSTEM elimina la sede
6. else if
    6.1. Lider selecciona en cancelar la eliminación
    6.2. SYSTEM cancela la eliminación y vuelve al modulo de gestionar las sedes
   end if

Escenario 4: busqueda
Steps
1. Lider selecciona en el botón gestionar las sedes en el menú principal
2. SYSTEM abre un modulo con todas las sedes existente
abre un modulo con todas las sedes existente
3. Lider selecciona en el botón consultar
4. SYSTEM abre una ventana donde se sólita las opciones para buscar la sede a su gusto
5. Lider digita los datos correspondientes y selecciona en consultar
6. SYSTEM valida y busca la sede

Extension
Steps
6.a. botón cancelar
    1. Lider cancela el proceso
    2. SYSTEM muestra la interfaz del modulo gestionar las sedes
